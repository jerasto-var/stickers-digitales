# 🎈 Generador de Stickers de Nombres

Este es un proyecto sencillo para generar archivos **SVG** con efecto de "globo" o "sticker". 
Ideal para crear logotipos rápidos, etiquetas personalizadas o recursos visuales vectoriales.

## ✨ Características
- **Efecto Globo:** Crea un diseño con volumen y contorno grueso.
- **Efecto Contorno:** Un diseño minimalista con espacio transparente.
- **Descarga Directa:** Exporta el archivo listo para usar en diseño web o impresión.
- **Optimizado para móvil:** Úsalo directamente desde tu celular.
